# nyein-terminal

`nyein-terminal` is my terminal portfolio website built with React and Typescript.

<img width="1440" height="812" alt="Image" src="https://github.com/user-attachments/assets/ecb668dd-8312-4bfe-9f0f-af355d2e3bfd" />

Create env file(Optional)

```bash
cp .env.example .env
```

Install Dependencies

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
